# 9IADT_TechChallenge_2

Este projeto contém experimentos de otimização aplicados a problemas médicos (Diabetes e Câncer de Mama).  
Os resultados incluem métricas de fitness, gráficos de evolução e interpretações médicas geradas com modelos da Hugging Face.
